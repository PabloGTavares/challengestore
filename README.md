## 📌 Propósito da análise realizada

O desafio tem como objetivo **identificar qual loja da Alura Store deve ser vendida** para investir em um novo negóco 
Para isso, realizamos uma **análise comparativa entre todas as lojas**, considerando indicadores/métricas essenciais como:

- 📊 **Faturamento total**
- ⭐ **Avaliações dos clientes**
- 🛒 **Categorias de produtos**
- 📈 **Desempenho de vendas**
- 🚚 **Frete médio**

A finalidade é **avaliar a eficiência de cada loja** e, com base nesses critérios, recomendar de forma fundamentada **qual delas deve ser vendida** para que o negócio se torne mais **sustentável e competitivo**.

## 📂 Estrutura do projeto e organização dos arquivos

O repositório está organizado de forma objetiva, contendo apenas os arquivos essenciais para o desafio:

- 📑 **README.md**  
  Documento principal que descreve o propósito da análise, a estrutura do projeto, exemplos de gráficos e insights obtidos, além das instruções para execução.

- 📓 **Desafio_AluraStoreBrasil.ipynb**  
  Notebook em Python que concentra toda a análise exploratória dos dados, geração dos gráficos e a conclusão fundamentada sobre qual loja deve ser vendida.

  ## 📊 Exemplos de gráficos e insights obtidos

Durante a análise foram produzidos diferentes gráficos que ajudaram a visualizar o desempenho das lojas e fundamentar a decisão final:

- 📊 **Gráfico de barras – Faturamento por loja**  
  *Insight:* A Loja 1 apresentou o maior faturamento, mas isso não garantiu eficiência, pois outros indicadores mostraram fragilidades.

- 📊 **Gráfico de barras – Média de avaliações dos clientes**  
  *Insight:* As Lojas 2 e 3 tiveram as melhores avaliações, enquanto a Loja 1 obteve a pior média, indicando baixa satisfação dos clientes.

- 📊 **Gráfico de barras empilhadas – Categorias mais e menos vendidas**  
  *Insight:* As lojas mais eficientes se destacaram em categorias estratégicas, enquanto a Loja 1 não mostrou vantagem competitiva clara.

- 📊 **Gráfico de barras – Produtos mais e menos vendidos**  
  *Insight:* A Loja 1 apareceu com maior presença entre os produtos encalhados, revelando baixo giro e estoque parado.

- 🍕 **Gráfico de pizza – Média de frete por loja**  
  *Insight:* A Loja 1 apresentou o frete mais caro, prejudicando a conversão de vendas; já a Loja 4 teve o frete mais baixo, favorecendo sua eficiência.
